## Project leírás

`A Beatwave egy innovatív platform, amely a zenei statisztikák elemzését ötvözi az interaktív közösségi élménnyel. A rendszer a Spotify-adatok alapján személyre szabott, dinamikusan frissülő toplistákat készít a felhasználók zenei szokásairól, emellett pedig egy gyors, Threads-szerű közösségi felületen lehetőséget biztosít a zenei témájú beszélgetésekre. A projekt fő célja, hogy a zenerajongók egyetlen helyen követhessék hallgatási statisztikáikat, miközben megosztják és megvitatják élményeiket másokkal.`
